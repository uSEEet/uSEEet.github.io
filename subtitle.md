### cali.
> 素材 181204flab-cali.mov   fusionLab 眼动仪校准过程

- 空白标题页:
    + 眼动仪校准
- 00:01
    + 点击 `1.执行 校准引导`
- 00:07
    + 弹出校准引导界面
    + 并自动打开摄像调整窗口
- 00:10
    + 首先调节眼睛摄像头
- 00:13
    + 尽力将眼球安置在图像中央
- 00:18
    + 并转动眼球
    + 确保在四角都能识别出瞳孔
- 00:21
    + 有红圈说明已识别
- 00:27
    + 再调整坐位和视野摄像头
    + 确保屏幕在图像中央
- 00:31
    + 点击 `去校准`
- 00:33
    + 弹出全屏界面
- 00:37
    + 校准期间头不得移动
    + 只能转动眼球盯住屏幕耙标中心点
- 00:42
    + 无论屏幕耙标移动到哪
    + 都要尽力不动头盯住
- 00:51
    + 有时中心点出现红圈
    + 说明瞳孔没有识别
- 00:55
    + 而外圈绿边消失
    + 说明系统没有识别耙标
- 01:31
    + 一共7 个校准点
- 01:35
    + 完成校准后
    + 系统自动给出是否通过的判定
- 01:38
    + 点击 `否` 退出校准
- 01:40
    + 点击 `检验` 进行人工校准效果检验
- 01:45
    + 实时视频中 兰圈白点 
    + 是当前系统转换的注目点
- 01:49
    + 通过注视测试物体
    + 来观察视频中注目点是否精确
- 01:55
    + 明确效果吻合预期后
    + 关闭相关窗口
- 01:57
    + 系统将自动保存校准成果
    + 用于接下来的操作



### recordings
> 素材 181204flab-reco.mov   fusionLab 多模实验记录过程

- 空白标题页:
    + 多模心理数据
    + 实时记录
- 00:01
    + 点击 `2.开始 实验记录`
- 00:06
    + 弹出记录控制器界面
- 00:08
    + 黑色区域是 EEG 实时数据流
- 00:11
    + 点击 `启动记录进程`
    + 启动相关后台程序
- 00:14
    + 观察到红色箭头提示符
    + 说明所有记录后台程序准备好了
- 00:17
    + 点击 `开始记录`
    + 弹出实验命名窗口
- 00:23
    + 注意, 只能使用数字和英文
    + 不能包含空格以及中文和其它特殊字符
- 00:35
    + 点击 `结束记录`
    + 准备结束本次记录
- 00:40
    + 如果界面没有反应
    + 可以多次尝试点击
- 00:46
    + 结束成功, 界面将自动完成按钮状态切换
- 00:55
    + 可以多次点击 `开始记录`
    + 为同一位被试进行多次实验记录
- 01:02
    + 对应 `结束记录`
    + 系统自动分目录保存不同记录
- 01:05
    + 点击 `关闭记录进程`
    + 结束所有后台有关程序
- 01:09
    + 观察到红色箭头提示符数量变化
- 01:13
    + 现在可以安全关闭窗口
